# Pokedex

## Descripción
🏋️‍♀ **Attentip** es una aplicación para dispositivos android que muestra ejercicios para realizar en una pausa activa. Se clasifican en ejercicios fisicos y en ejercicios mentales.

## Características
* Fragments
* Material Design
* Shared Preferences
* Topbar
* Dialog Fragment
* Recycle View
